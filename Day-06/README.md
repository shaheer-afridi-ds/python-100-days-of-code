# Python 100 Days of Code - Day 6

## Overview
Day 6 focuses on **Functions, Loops, and Challenges**.  
You will learn how to define and call functions, use loops (`for` and `while`), and solve simple challenges like hurdles and mazes.  

This day helps in building **problem-solving skills** using Python in a beginner-friendly way.

---

## Topics Covered
1. **Defining and Calling Functions**
   - Using `def` to create functions
   - Calling functions
   - Examples: greeting, adding numbers, saying goodbye

2. **Built-in vs User-defined Functions**
   - `print()`, `input()`, `len()` → Built-in
   - Custom functions like `greet()` → User-defined

3. **Indentation in Python**
   - Python uses spaces to define code blocks
   - Examples with `if`, `for`, `while`, and functions

4. **While Loops**
   - Running code repeatedly while condition is true
   - Examples: counting, Reeborg hurdles

5. **Hurdle Challenges**
   - Using loops and functions to jump over hurdles
   - Challenges with **fixed** and **variable heights**

6. **Final Project: Escaping the Maze**
   - Combining loops, functions, and logic
   - Follow right-hand wall strategy
   - Reach the goal step by step

## Learning Points
- Functions make code **reusable** and organized  
- Loops help to **repeat tasks** efficiently  
- Indentation is **critical** in Python  
- Real beginner projects like **hurdles and maze** improve problem-solving  
- Step-by-step coding helps in building **confidence for bigger projects**
